===========================
Instruction to Run Project
===========================

How to Run Project

Unzip the file blood_donation.zip on your system copy blood_donation folder to C:\xampp\htdocs



Database Configuration :- 

Open phpmyadmin “http://localhost/phpmyadmin”
Create Database bd
Import database bd.sql
Open Your browser put inside browser "http://localhost/blood_donation/index.php"
For Admin Panel
Open Your browser put inside browser http://localhost/blood_donation/admin/login.php
